# 機能概要

## 💰 価格監視
* ゲーム価格一覧・リアルタイム更新・価格推移グラフ
* 価格/割引率閾値アラート・Discord通知・履歴管理
* 未リリースゲーム追跡・自動リリース検知

## 🎮 ゲーム管理
* CRUD操作・Steam連携・画像表示・タイプ自動検知
* 購入状況追跡・手動最安値設定・統一UI

## 💼 出費追跡・予算管理
* タブ式統合ダッシュボード・Canvas APIチャート・ROI分析
* 月間/年間/カスタム予算設定・リアルタイム監視・アラート
* 支出パターン分析・投資効率指標・統計ダッシュボード

## 🆓 無料ゲーム監視
* Epic Games & Steam無料ゲーム統合検知
* RSS監視 (1時間間隔)・受け取り状況管理・自動ゲーム登録
* Discord通知・統計情報・WebUI専用ページ

## 🔧 システム機能
* **異常検知**: 平均購入価格2倍超、月間支出50%以上増加検知
* **予算関連アラート**: 使用率警告、超過通知
* **節約マイルストーン**: 目標達成祝福通知
* **アラート管理**: 重要度分類、未読/既読状態、詳細表示

### ダッシュボードカスタマイゼーション
* **レイアウト管理**: ウィジェット表示/非表示、サイズ調整（小/中/大/全幅）
* **テーマシステム**: 3つのカラーテーマ（ブルー/グリーン/ダーク）、ライト/ダーク/自動モード
* **ユーザー設定**: デフォルトビュー、自動更新間隔、通知設定
* **設定永続化**: localStorage自動保存・復元

### レポート生成システム
* **複数フォーマット**: PDF/CSV/JSON出力対応
* **カスタマイズ可能セクション**: サマリー、チャート、ROI、予算、詳細テーブル
* **リアルタイムプレビュー**: Markdownスタイル表示
* **期間設定**: 月間/年間/カスタム/総合レポート

### データ管理システム
* **バックアップ機能**: 完全/設定のみ/出費データ/予算データの4種類
* **インポート/エクスポート**: JSON形式での設定移行・データ共有
* **データ整合性**: チェックサム検証による破損検知
* **安全な復元**: 確認ダイアログとページリロード

## Epic Games統合機能

### 無料ゲーム検知システム
* **Epic Games Store連携**: epic-free-games パッケージによる自動検知
* **リアルタイム監視**: 24時間間隔での自動チェック（カスタマイズ可能）
* **複数データソース**: Epic Games API + IsThereAnyDeal API フォールバック
* **正確な期限管理**: promotionalOffers/upcomingPromotionalOffers からの正確な終了日取得

### 受け取り管理機能
* **受け取り状況追跡**: 個別ゲームの受け取り済み/未受け取り状況管理
* **受け取り状況切り替え**: ワンクリックでの受け取り済み状態トグル機能
* **フィルタリング機能**: 全て/配布中/未受け取り/受け取り済みでの絞り込み表示
* **統計情報**: 配布中ゲーム数、未受け取り数、受け取り済み数の可視化

### Discord通知連携
* **リッチ埋め込み通知**: 新規無料ゲーム発見時の自動Discord通知
* **詳細情報表示**: ゲーム画像、説明、Epic Store直接リンク
* **期限情報**: 残り時間の日本語表示（「あと3日」形式）

### Epic Games専用UI
* **専用ページ**: Epic Games無料ゲーム管理専用インターフェース
* **カード式レイアウト**: ゲーム画像・詳細情報・アクションボタン統合表示
* **手動更新機能**: 「最新データ取得」ボタンによる即座更新
* **Epic Store統合**: 各ゲームへの直接リンク機能

## 高割引ゲーム検知機能

### ITAD統合システム
* **高割引ゲーム自動検知**: IsThereAnyDeal API直接利用による包括的割引ゲーム取得
* **動的フィルタ設定**: 最小割引率・最大価格・取得件数・地域設定の柔軟な設定
* **リアルタイム検知**: スケジューラー連携による定期的な高割引ゲーム監視
* **Discord通知統合**: 高割引ゲーム発見時の自動通知システム

### 高割引ゲーム表示システム
* **専用コンポーネント**: HighDiscountGames.tsx による統合表示
* **ページネーション対応**: 大量ゲーム対応・ページ分割表示（50件/ページ）
* **詳細情報表示**: 価格・割引率・Steam/SteamDB/Backloggd直接リンク
* **購入連携**: 高割引ゲームから直接購入記録・予算連携

### ITAD設定管理システム
* **専用設定ページ**: /settings/itadタブによる直感的設定インターフェース
* **データベース連携**: itad_settingsテーブルによる永続的設定保存
* **設定カテゴリ管理**: filter/general/notification カテゴリ別設定組織化
* **リアルタイム反映**: 設定変更の即座反映・検証機能

## 購入管理システム

### 統合購入記録
* **専用購入モーダル**: PurchaseModal.tsx による独立した購入記録UI
* **予算連携機能**: 購入時の予算指定・自動支出記録システム
* **購入情報分離**: 編集モーダルから購入関連機能完全分離・UX向上
* **ワンクリック購入**: ゲーム一覧からの直接購入記録・統合管理

### 購入データ管理
* **購入履歴追跡**: 購入価格・日付・使用予算の包括的記録
* **ROI分析統合**: 購入データと出費分析システムの完全連携
* **購入統計**: 購入済みゲーム数・総支出・平均価格の可視化
* **予算配分**: 購入時の予算自動減算・残高リアルタイム更新

## システム機能

### データ管理
* 包括的バックアップシステム (エクスポート/インポート)
* データベースマイグレーション機能
* 設定状況確認・ヘルスチェック

### UI/UX機能
* **React SPA**: React 19.1.0 + TypeScript + Vite
* **レスポンシブデザイン**: デスクトップ・タブレット・モバイル対応
* **ダークモード**: システム設定追従・視認性最適化
* **統一ナビゲーション**: 全ページ共通インターフェース
* **レイアウト最適化**: 大きな金額表示での改行問題解決（clamp()使用のレスポンシブフォントサイズ）
* **バッジ表示修正**: 条件達成後の謎の「0」表示エラー修正（!!演算子による型変換）
* **統計カード統一**: 全統計カードで一貫したフォントサイズとレイアウト

## 設定・テスト機能

### 統合設定システム
* **専用設定ページ**: /settings による統合設定管理インターフェース
* **タブ式設計**: ITAD設定・通知設定・監視設定・API設定の分離組織化
* **将来拡張対応**: モジュラー設計による新機能設定追加容易性
* **設定永続化**: データベース連携による設定値自動保存・復元

### Discord連携テスト
* **6種類のテスト機能**: simple/rich/price-alert/error/game-alert/spending-alert
* **リッチEmbed対応**: カスタム埋め込み・接続状態確認
* **価格アラートシミュレーター**: 実際のゲームデータを使用したテスト送信
* **カスタムメッセージ**: 任意メッセージによるWebhook接続確認

### システム管理
* **設定ダッシュボード**: 統合設定画面・リアルタイム状況監視
* **APIキー管理**: 6種類のAPI設定状況可視化・バッジ表示
* **ビルド情報システム**: 実際のビルド日時・バージョン・環境情報表示
* **システム情報**: メモリ使用量・稼働時間・プラットフォーム情報

## ユーザビリティ仕様

* **レスポンシブ対応**:
    * デスクトップ (1920px以上): 3カラム。
    * タブレット (768px-1919px): 2カラム。
    * モバイル (767px以下): 1カラム、スタック表示。
* **ダークモード対応**:
    * ヘッダーにトグルボタン。
    * localStorageで設定永続化。
    * システム設定に追従オプション。
* **キーボードショートカット**:
    * `Ctrl/Cmd + R`: 手動更新
    * `Ctrl/Cmd + A`: ゲーム追加モーダル
    * `Ctrl/Cmd + S`: 設定画面
    * `Ctrl/Cmd + D`: ダークモード切り替え
    * `ESC`: モーダル・ドロップダウン閉じる
* **アクセシビリティ**:
    * 色覚異常対応、コントラスト (WCAG 2.1 AAレベル準拠)。
    * スクリーンリーダー (aria-label, role属性)。
    * フォーカス (タブ移動でキーボード操作可能)。
